# My Godot Action RPG Game

This is a simple 2D action-adventure game built using **Godot Engine 4**.  
The game includes basic combat, NPC interactions, dialogue systems, multiple scenes,  
and transition logic between world areas such as the Main House, Cliffside, and Dungeon.
